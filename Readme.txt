System requirements 
  Memory: 8GB (recomended 16GB) 
  VCPU: 4 (Recomended 8)
  Volume: 32GB 

Run Druid in Docker container
  Pre-requisites: Docker & Docker-compose
  Install Docker & Docker-compose using the below link in Centos/RHEL/Debain/Ubuntu
  https://github.com/devopslife999/Setup_Quick_Access/tree/main/Docker
  
  For Centos/RHEL linux execute "sudo docker compose up -d" to run Druid in docker container
  For Debain/Ubuntu linux execute "sudo docker-compose up -d" to run Druid in docker container
  Access Druid at 8888 port "localhost:8888 or <publicip>:8888"
